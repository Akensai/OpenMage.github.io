Supported Versions

The initial Branch 1.9.4.x is aligned to the Official Magento Release. 20.0 will be the first Magento independent Release we do. Every 4 Years there will be an LTS version, which gets 5 years of security support.

Disclaimer: the timeframes are still open for discussion.
The related RFC process is currently in Draft state.
Participation of any OpenMage user is extremely welcome.
